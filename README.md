# Aplicación Web para la Gestión de Imprenta

🚧 **Proyecto en desarrollo (MVP – ~60%)**

Aplicación web desarrollada para digitalizar la gestión de una imprenta gráfica que actualmente maneja su información de forma manual, permitiendo centralizar y organizar los datos del negocio mediante una plataforma web.

---

## 📌 Descripción del proyecto

Esta aplicación busca solucionar la gestión manual de una imprenta gráfica, digitalizando procesos como el registro de clientes, productos y cotizaciones.  
El sistema permite un mejor control del negocio mediante autenticación, base de datos y acceso centralizado desde la web.

---

## 🧩 Módulos del sistema

### ✅ Implementados
- **Autenticación (Login)** con JWT
- **Gestión de Productos**
  - Registro
  - Listado
  - Subida de imágenes
- **Gestión de Clientes**
  - Registro
  - Edición
  - Listado

### 🚧 Pendientes
- cotizaciones
- Cobranzas
- Contabilidad
